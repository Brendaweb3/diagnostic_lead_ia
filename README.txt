# Diagnostic Lead IA

**Diagnostic Lead IA** est une application Streamlit qui permet d’évaluer rapidement la maturité digitale et commerciale de votre entreprise, de mesurer votre “friction IA” et d’éditer un mini-audit personnalisé accompagné de conseils, en PDF. L’application offre aussi la prise de rendez-vous directe pour des recommandations sur-mesure.

## Fonctionnalités

- **Formulaire interactif** avec questions sur qualification des leads, automatisation, intégration IA, priorités business…
- **Calcul automatique** du score de friction IA (sur 45 points)
- **Audit détaillé et conseils personnalisés** selon votre profil et vos réponses
- **Téléchargement d’un PDF** récapitulatif avec branding et synthèse
- **Lien Calendly** intégré pour réserver un échange découverte
- **Interface simple, RGPD-Friendly, zéro code à installer**

## Installation

Prérequis :
- Python 3.7+
- pip

Installation rapide :
```bash
pip install -r requirements.txt
```

## Lancement de l’application

Dans le dossier du projet :
```bash
python -m streamlit run main_final_brendabox.py
```
Puis : ouvrez le lien http://localhost:8501 dans votre navigateur.

## Structure du projet

- `main_final_brendabox.py` : script principal Streamlit
- `requirements.txt` : dépendances à installer (streamlit, reportlab)
- `assets/logobrendabox.png` : logo pour le PDF (à personnaliser)
- `README.md` : ce fichier d’aide
